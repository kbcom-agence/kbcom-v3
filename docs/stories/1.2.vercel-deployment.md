### GitHub Actions CI Workflow

[Source: architecture.md#CI/CD Pipeline]

**Workflow File:** `.github/workflows/ci.yml`

**Trigger:** On pull_request to main, develop branches

**Jobs:**

1. **Lint & Type Check**
   - Run ESLint
   - Run TypeScript type checking
   - Fail if errors found

2. **Test**
   - Run Vitest unit tests
   - Upload test coverage (optional)

3. **Build**
   - Test production build
   - Ensure no build errors

**Required Configuration:**

```yaml
name: CI

on:
  pull_request:
    branches: [main, develop]

jobs:
  lint-type-test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: '20'
          cache: 'npm'

      - name: Install dependencies
        run: npm ci

      - name: Run ESLint
        run: npm run lint

      - name: Type check
        run: npm run type-check

      - name: Run unit tests
        run: npm test -- --run

      - name: Build
        run: npm run build
```

### Lighthouse CI Configuration

[Source: architecture.md#CI/CD Pipeline, NFR Performance Requirements]

**Tool:** @lhci/cli (Lighthouse CI)

**Configuration File:** `lighthouserc.json`

**Performance Budgets (from NFR):**

- Lighthouse Performance: ≥ 95
- Lighthouse Accessibility: ≥ 95
- Lighthouse Best Practices: ≥ 95
- Lighthouse SEO: 100
- Largest Contentful Paint (LCP): < 1.2s
- First Input Delay (FID): < 100ms
- Cumulative Layout Shift (CLS): < 0.1
- Total Blocking Time (TBT): < 200ms

**lighthouserc.json Example:**

```json
{
  "ci": {
    "collect": {
      "startServerCommand": "npm run build && npm run start",
      "url": ["http://localhost:3000"],
      "numberOfRuns": 3
    },
    "assert": {
      "preset": "lighthouse:recommended",
      "assertions": {
        "categories:performance": ["error", { "minScore": 0.95 }],
        "categories:accessibility": ["error", { "minScore": 0.95 }],
        "categories:best-practices": ["error", { "minScore": 0.95 }],
        "categories:seo": ["error", { "minScore": 1.0 }],
        "largest-contentful-paint": ["error", { "maxNumericValue": 1200 }],
        "cumulative-layout-shift": ["error", { "maxNumericValue": 0.1 }],
        "total-blocking-time": ["error", { "maxNumericValue": 200 }]
      }
    },
    "upload": {
      "target": "temporary-public-storage"
    }
  }
}
```

**GitHub Actions Workflow:** `.github/workflows/lighthouse-ci.yml`

```yaml
name: Lighthouse CI

on:
  pull_request:
    branches: [main, develop]

jobs:
  lighthouse:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: actions/setup-node@v4
        with:
          node-version: '20'

      - name: Install dependencies
        run: npm ci

      - name: Run Lighthouse CI
        run: |
          npm install -g @lhci/cli
          lhci autorun
        env:
          LHCI_GITHUB_APP_TOKEN: ${{ secrets.LHCI_GITHUB_APP_TOKEN }}
```

**Note:** Initial Lighthouse scores may be lower than targets. Optimization happens in Epic 7 (Story 7.x).
