### Metadata Generation Structure

[Source: prd.md#Story 1.3 Technical Notes]

**generatePageMetadata Helper Example:**

```typescript
// lib/seo/metadata.ts
export function generatePageMetadata(page: {
  title: string;
  description: string;
  path: string;
  keywords?: string[];
}) {
  return {
    title: `${page.title} | KB-COM`,
    description: page.description,
    keywords: page.keywords,
    openGraph: {
      title: page.title,
      description: page.description,
      url: `https://kb-com.fr${page.path}`,
      siteName: 'KB-COM',
      locale: 'fr_FR',
      type: 'website',
    },
    alternates: {
      canonical: `https://kb-com.fr${page.path}`,
    },
  };
}
```

**Key Requirements:**

- French locale: `fr_FR`
- Domain: `https://kb-com.fr`
- Title template: Include "KB-COM" branding
- Canonical URLs for all pages
- OpenGraph for social sharing
- Keywords optional (Google ignores, but good for documentation)

```typescript
export const metadata: Metadata = {
  title: {
    template: '%s | KB-COM',

  },
  description: 'Agence Web à Tours...',
  metadataBase: new URL('https://kb-com.fr'),
  openGraph: {
    // ...config
  },
};
```

**Metadata in Page Components:**

```typescript
import type { Metadata } from 'next';

export const metadata: Metadata = {
  title: 'Services',
  description: 'Nos services...',
};
```

**Dynamic Metadata:**

```typescript
export async function generateMetadata({ params }): Promise<Metadata> {
  return {
    title: params.title,
    // ...
  };
}
```
