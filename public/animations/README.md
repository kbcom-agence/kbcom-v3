# Animations Rive

Ce dossier contient les fichiers d'animation Rive (.riv) utilisés dans la section Services.

## Fichiers requis

Placez vos animations Rive exportées ici avec les noms suivants :

- `website-animation.riv` - Animation pour le service "Sites Web"
- `ecommerce-animation.riv` - Animation pour le service "E-Commerce"
- `seo-animation.riv` - Animation pour le service "SEO & Référencement"
- `app-animation.riv` - Animation pour le service "Applications"

## Ressources

- **Créer des animations** : [Rive Editor](https://rive.app/)
- **Communauté** : [Rive Community](https://rive.app/community/)
- **Assets gratuits** : [Rive Community Files](https://rive.app/community/files/)

## Notes

Les animations doivent :
- Être légères (< 500KB recommandé)
- Avoir un "State Machine" nommé "State Machine 1" pour l'autoplay
- Être en boucle pour une lecture continue
- Être optimisées pour le web
